# MarketLab Workshop Checkpoints

MarketLab starts intentionally small. Build the workshop in checkpoints so each step has one clear concept and a working app state.

## Step 1: Email/password auth

Add the email/password sign-up, sign-in, and sign-out path with Supabase Auth.

## Step 2: Profiles and fake starting balances

Create owner-scoped profiles and assign fake starting balances in cents.

## Step 3: Seeded markets and public market pages

Add fictional, non-political seed markets and public pages that anyone can read.

## Step 4: Trade action/RPC

Implement the trade Server Action and Supabase RPC for balance-changing operations.

## Step 5: Portfolio and settlement

Show each user's positions, then add a simple settlement path for workshop markets.
